# Nodesappy
Nodesappy is a boilerplate for node.js to create modern web app

## Integrated

- Tailwindcss
- Reactjs
- DaisyUI
- expressjs
- authjs
- vitejs
- postcss
## Get started
First ,clone this repo to your local.

Second ,open it,and open the cmd of it.Use "npm i" to download the packages.
